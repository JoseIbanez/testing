vRealize Build Tools Workstation

vRealize Build Tools Workstation represents a vRealize engineer development machine configured to work with vRealize Build Tools toolchain.

Install and Configure

Prerequisites

  • vRealize Build Tools Platform ready to use
  • vRealize Developer Tools
  • Java 8 (official installation guide)
  • Maven 3.5+ (official installation guide)
  • Development vRealize Automation Tenant configured with development vRealize Orchestrator
    • Tenant administrator user
    • Workstation can access vRA server on port 443
  • Development vRealize Orchestrator
    • vRO administrator user
    • vRO appliance root user
    • Workstation can access vRO server on ports [443 or 8281], 8283
